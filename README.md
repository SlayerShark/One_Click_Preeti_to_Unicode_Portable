Preeti to Unicode for MS-Word - Portable

For Nepali typists still using the typical Preeti font layout, which isn’t compatible with the web interface, I developed a portable tool that converts text to Unicode with a single click in Microsoft Word, no need to copy it to the web interface and paste it back. It works hassle‑free; just open the tool and run it alongside Word.

## Usage Instructions  
1. Open your target document in Microsoft Word.  
2. Select the Preeti text block you want to convert.  
3. Run WordPreetiToUnicode.exe with the document open.  
4. Click \*\*Convert Selected Text\*\* to start the conversion.
